Mountbatten Maps

From here you can explore a few mapping experiments. Some will develop into proper maps. If I invited you to this page, I did that because I really appreciate feedback.

  • Simple map showing modernist houses in Ng'ambo. Built with the Leaflet.MapboxVectorTile.js> layer built by SpatialDev. Great job guys. Thanks for everything.
  • Standard sample of vectortiles completely copied from the Mapzen samples. Uses the MapBox javascript library.
  • Full blown map of building age of all buildings in The Netherlands. There are 10M buildings in The Netherlands. Based on Mapzen's Tangram library.
This is very much work in progress so follow me on Twitter or let's have a beer so I can tell you where this is heading.
